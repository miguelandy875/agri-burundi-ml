# Prédiction des Récoltes au Burundi — ML Pipeline

Ce projet met en place un pipeline de Machine Learning pour analyser des données agricoles simulées du Burundi et prédire si une récolte sera bonne ou mauvaise à partir de facteurs agronomiques, climatiques et géographiques.
